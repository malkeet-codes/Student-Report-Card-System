# Student-Report-Card-System
## 📌 Description
This project is a simple system that generates a student report card based on marks entered by the user. It calculates the total marks, percentage, and overall performance.

## 🚀 Features
- Input marks for multiple subjects
- Automatic total and percentage calculation
- Performance evaluation (grade/result)
- Simple and user-friendly interface

## 🛠️ Technologies Used
- Python

## 🎯 Purpose
This project was created to practice basic programming concepts such as user input, calculations, and conditional logic while simulating a real-world academic system.
